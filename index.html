Cariflex 관리자 페이지

프로젝트 개요

관리자 페이지를 신규 구축하여 계정 관리·로그 관리·게시판 등록 기능 등을 퍼블리싱. 내부 운영 담당자가 콘텐츠를 직접 관리할 수 있도록 하는 관리자 전용 웹 인터페이스.

담당 역할 및 기여도
  • - 관리자 페이지 전체 퍼블리싱 100% (기획 단계부터 구조 설계 참여)
  • - 로그인, 관리자 계정 관리, 접속 로그/권한 로그 관리 화면 퍼블리싱
  • - 게시판(공지/소식글) 등록 및 리스트 화면 구현 (HTML 기반 주고 설계 및 JavaScript 인터랙션 처리)
  • - 날짜 선택 기능 구현(Date Picker 활용) 및 UX 흐름 개선 제안
기술 스택
