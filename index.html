ABOUT ME

현재 6년차 웹 퍼블리셔로 인하우스 운영 중심의 실무를 맡아왔습니다.
JSP 기반부터 React 환경까지 다양한 개발 구조 속에서 퍼블리싱을 경험했고

인하우스 운영과 유지보수 경험을 통해 기획자·디자이너·개발자와의 협업에 익숙하며,
타인의 코드를 빠르게 파악하고 오류를 수정하는 데 강점을 가지고 있습니다.

또한 JavaScript(ES6)를 활용해 인터랙션과 동작을 직접 구현할 수 있으며,
유틸리티 클래스 기반의 레이아웃 설계(Tailwind CSS 스타일)에 익숙합니다.
